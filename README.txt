This is a Tetris game written in C++ with ncurses library. 
The library only works with Linux environment, therefore, please run the program in Linux by typing ./Tetris

The game include basic mechanism of a ordinary Tetris game. The game keeps the player's score, and it also supports saving the highest five socres ever reached.
Operation guidance: Use arrow keys to move and rotate blocks, and press "p" to pause or unpause the game. The game will end automaticlly if a block touches the top of the screen.
